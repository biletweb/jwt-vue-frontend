## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

To work you need to install the frontend <a href="https://github.com/biletweb/jwt-laravel-backend">jwt-laravel-backend</a>.
